# Simple loan calculator

This task is created by blueberry.io, see
https://js-developer-second-round.herokuapp.com/.

The devstack is based on https://github.com/tajo/devstack.git.

## Download

```shell
git clone https://github.com/pavelzima84/loan-calculator
cd loan-calculator
yarn install
```

## Development mode

```shell
npm start
open http://localhost:8000
```

## Production mode (not tested yet)

```shell
npm run build
npm run server
open http://localhost:8000
```

## ESLint

```shell
npm run lint
```
